# comandWork
Формирование навыков разработки клиент-серверных приложений в условиях командной работы.

Задача: Реализовать основной функционал интерактивной доски, включая создание, редактирование, перемещение, удаление и синхронизацию графических и текстовых объектов между несколькими пользователями.
